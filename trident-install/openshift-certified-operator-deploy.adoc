= Trident 커뮤니티 운영자에서 OpenShift 인증 운영자로 전환하세요
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp 커뮤니티 Trident Operator에서 Red Hat OpenShift 인증 Trident Operator로 전환하려면 커뮤니티 Operator를 제거한 다음 OperatorHub를 사용하여 인증 Operator를 설치해야 합니다.

.시작하기 전에
설치를 시작하기 전에 link:../trident-get-started/requirements.html["Trident 설치를 위한 환경을 준비하세요"].



== NetApp Trident 커뮤니티 운영자를 제거합니다

.단계
. OpenShift 콘솔을 사용하여 OperatorHub로 이동합니다.
+
image::../media/openshift-operator-05.png[설치]

. NetApp Trident 커뮤니티 운영자를 찾으세요.
+
image::../media/openshift-operator-06.png[설치됨]

+

WARNING: *이 연산자에서 모든 피연산자 인스턴스 삭제*를 선택하지 마십시오.

. *제거*를 클릭합니다.




== OpenShift 인증 운영자를 설치하세요

.단계
. Red Hat OperatorHub로 이동합니다.
. NetApp Trident Operator를 검색하여 선택하십시오.
+
image::../media/openshift-operator-05.png[설치]

. 화면의 지시에 따라 Operator를 설치하십시오.




== 확인

* 콘솔의 OperatorHub에서 새 인증 운영자가 성공적으로 설치되었는지 확인하십시오.
